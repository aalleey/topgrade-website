Languages and Technologies used in TopGradeSoftware Website:

1. HTML5: Used for the structural foundation of all platform pages (index.html, pos.html, sale.html, etc.).
2. CSS3 (Vanilla CSS): Used for the high-end, premium design system, including custom theme variables, animations, and responsive layouts.
3. JavaScript (ES6+): Used for dynamic theme management, reusable UI components (footer-component.js), and interactive client-side logic.
4. Markdown: Used for documentation (README.md).

PAGE DETAILS
--------------------------------------------------
• Home (index.html): Central gateway showcasing overall capabilities, brand identity, and featured software solutions.
• Solutions Hub (sale.html): Comprehensive listing of all enterprise software with pricing tiers and a lead conversion demo form.
• Themed Dashboard (dashboard.html): Allows live testing and switching between 10+ premium themes (Cybernova, Midnight, Sunset, etc.).
• Industry Pages: Specialized pages for each vertical (POS, Hospital, etc.) highlighting niche features and business benefits.
• Theme Manager: Centralized system responsible for the "Premium White" redesign and high-contrast accessibility.

Architecture Notes:
- Theme Engine: JavaScript-based CSS variable injector for real-time brand switching.
- Component System: Modular scripts for maintaining site-wide consistency (e.g. Footer).
- Modern Design: Implements glassmorphism, depth layers, and theme-aware shadows.
